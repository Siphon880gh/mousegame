Mouse Engine API Example: Alternate Actions

Open DevTools console, then perform different mouse clicks on this webpage to see the output.

Expected behaviors:

  • Single left click will console log LMB default action.
  • Single right click will console log RMB default action.
  • Holding RMB will show the alternate actions message. Release the RMB.
  • Single left click will console log LMB alternate action 1. Alternate actions message closes.
  • OR: Single right click will console log RMB alternate action 2. Alternate actions message closes.
  • Alternate actions are only possible when the alternate actions message is visible. Hold RMB to cancel the alternate actions message if prefer to perform neither alternate actions.
Alternate actions available
Press and hold RMB again to return to normal actions.
  • LMB for alternate action 1
  • RMB for alternate action 2
  • Hold RMB then release to cancel (neither actions will be performed)
